# Interview Question Vault 📋

**AI-powered tool to generate tailored interview questions** for QA, Business Analyst, and Telecom professionals.

## ✨ Features
- **Topic Mode**: Enter any topic (e.g., "OSS/BSS Testing", "Postman API Automation") → Get relevant questions
- **JD Mode**: Paste a full Job Description → Get highly specific, targeted questions
- Built with **Streamlit** + Grok (xAI)
- Download questions as TXT
- Focused on Software Testing, Telecom, and BA roles

## 🚀 Quick Start

### Local Run
```bash
# Clone the repo
git clone https://github.com/manyfacedhuman/interview-question-vault.git
cd interview-question-vault

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run interview_question_vault.py
